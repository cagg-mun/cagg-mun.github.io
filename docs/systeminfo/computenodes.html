  • Network -> iLO Dedicated Network Port -> SNTP (Primary Server 192.168.1.254, Time Zone America/St_Johns)
  • Power Management -> Server Power -> Uncheck “Auto Power-On”
  • Administration -> Access Settings -> Serial Command Line Interface Speed
